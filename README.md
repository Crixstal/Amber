## Features
- Sandstorm (with fog)
- Quicksands
- Animal life (giant worms, geckos)
- Footprints
- Other effects (sand decals, sand trails...)

## Controls
The controls are made for keyboard only:
- WASD - Move the player
- Mouse movements - Rotate the player
